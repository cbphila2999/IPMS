Welcome to the IPMS iFrame Integration Tab

Purpose:

This tab displays web-based tools and resources inside IPMS. Use it to view dashboards, help guides, training materials, or embedded apps.

How to Use

  1. Click "Add Link" to open the iFrame setup dialog.
  2. Paste a full web address starting with https://.
  3. Click "Save". The content will display below the tab name.

Helpful Examples

  • Embed a PowerBI Dashboard:
    https://app.powerbi.com/view?r=abc123
  • Link to a training video:
    https://www.youtube.com/watch?v=xyz789
  • Share a guidance document:
    https://heifer.sharepoint.com/sites/IPMS/Shared%20Documents/IPMS_Guide.pdf

Troubleshooting

  • If the page is blank, the external site may block iFrame embedding.
  • Only HTTPS links are allowed.
  • Try opening the link in a new browser tab to test access.
